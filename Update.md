# Update: ECS Integration And Primitive Rendering

## Что было сделано

- проект переведен с двух окон на одно окно с выбором активного рендера через конфиг
- загрузка `app.json` исправлена так, чтобы смена рендера применялась без новой пересборки
- с нуля внедрена ECS-архитектура: `Entity`, `World`, хранение компонентов, удаление сущностей, системы
- добавлены компоненты `Transform`, `Tag`, `MeshRenderer`
- добавлены служебные компоненты для движения и отскока
- ECS встроена в главный цикл приложения
- добавлен отдельный `RenderSystem`, который рисует ECS-сущности через существующий `RenderAdapter`
- сцена полностью переведена на ECS, старая отдельная логика треугольника убрана
- добавлены несколько ECS-объектов на сцену
- добавлены движение, вращение, отскоки и runtime `spawn/destroy`
- параметры ECS-сцены вынесены в `app.json`
- добавлена поддержка примитивов `line`, `triangle`, `quad`, `cube`
- добавлен цвет из `MeshRenderer` в DX12 и Vulkan
- архитектура подогнана под ТЗ: базовый `Component`, `Transform` в формате `position/rotation/scale`, `MeshRenderer`, `RenderSystem`, список систем в `World`

## Итог

На текущем этапе проект использует ECS как основную систему сцены и отрисовки объектов, а стартовая сцена и ее параметры задаются через конфиг.
